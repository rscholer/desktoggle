==========
desktoggle
==========

-------------------------------------
hide all windows and show the desktop
-------------------------------------

:Author: Raphael Scholer <rascholer@gmail.com>
:Manual group: User Commands
:Manual section: 1

SYNOPSIS
--------
``desktoggle`` [*OPTION*]...

DESCRIPTION
-----------
Toggle between hiding and showing of all windows.

OPTIONS
-------
--help
  Show usage information.

--version
  Show version information.
